Demon List Visualised

Welcome to Demon List Visualised! This is personal project of mine to improve at programming. The purpose of this website is to graphically represent relationships and trends of the hardest levels in Geometry Dash.

Simply select the values for the x-axis and y-axis, then press 'Create Graph'. Using data from the Global Demonlist and from Youtube, a graph will be plotted according to the chosen axes for the Top 75 hardest levels in Geoemetry Dash.
